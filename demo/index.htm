3.1  Headers

You always saw type 1 and 2 headers in action.

Type 3 headers

More than 3 types of headers are not recommended. It is a good typographical convention to have headers of a type 3 and more unnumbered.

3.3  Annotations

Example below shows a class member function annotation.

GraphicsDriver::DrawLine

Declaration

GraphicsDriver::Status GraphicsDriver::DrawLine(
    int x1, int y1, int x2, int y2,
    Color & color = Color::black
);

Description

Draw line from point to point.

Parameters

int x1 source x-coord
int y1 source y-coord
int x2 destination x-coord
int y2 destination y-coord
Color & color line color

Returns

GraphicsDriver::Status::OK if operation was successfuly completed. See GraphicsDriver::Status.

Exceptions

None.
